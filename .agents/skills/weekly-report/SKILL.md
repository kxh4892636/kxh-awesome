# 业界动态周报生成器

本 skill 用于生成电商业界动态与编程技术领域的周报。核心思路是：从指定的数据源逐一抓取最新内容，按分类整理，最终输出一份结构清晰的 Markdown 周报。

## 内容范围

- 时间窗口：仅包含最近两周内的动态
- 业界动态：优先治理体验相关的规则变更
- 技术动态：优先 AI、全栈、前端领域的最新进展
- 每个章节最多包含 13 条动态

## 工作流程

按以下顺序执行，每个数据源独立抓取，互不依赖，可并行处理。

### 第一步：抓取电商业界动态

逐一访问以下数据源，提取最近两周内的规则动态：

#### 淘宝规则

- URL: https://rulechannel.taobao.com/#/rules
- 操作：使用 WebFetch 抓取页面内容，找到【规则辞典】中的规则列表
- 筛选：提取最近两周内新增或变更的规则，重点关注与治理体验相关的内容
- 如果页面为 SPA 无法直接抓取，尝试使用 WebSearch 搜索"淘宝规则 最新更新"获取相关信息

#### 天猫规则

- URL: https://rulechannel.tmall.com/#/rules
- 操作：使用 WebFetch 抓取页面内容，找到【规则辞典】中的规则列表
- 筛选：同淘宝规则
- 备选：如果无法直接抓取，使用 WebSearch 搜索"天猫规则 最新更新"

#### 京东规则

- URL: https://learn-jdm.jd.com/knowledge/rule
- 操作：优先读取【每周规则变动一览】中的最新两篇文章
- 备选：如果无法读取变动一览，则读取【规则更新公示】中的规则列表
- 备选搜索：WebSearch 搜索"京东规则 最新更新"

#### 抖音电商规则

- URL: https://school.jinritemai.com/doudian/web/rules/11689
- 操作：读取最新的【本周新规速递】
- 备选搜索：WebSearch 搜索"抖音电商规则 本周新规"

### 第二步：抓取技术动态

逐一访问以下数据源，提取最近一周内的热门内容：

#### GitHub Trending

- URL: https://github.com/trending?since=weekly
- 操作：使用 WebFetch 抓取页面，提取本周热门项目列表
- 筛选：优先选择 AI、全栈、前端相关的项目
- 每个项目提取：项目名称、链接、简短描述、星标数

#### ByteTech

- URL: https://bytetech.info/articles?sort=HOTTEST
- 操作：使用 WebFetch 抓取页面，提取最新热门文章
- 筛选：优先选择 AI、全栈、前端相关的文章
- 每篇文章提取：标题、链接、简短概述

#### 掘金

- URL: https://juejin.cn/hot/articles
- 操作：使用 WebFetch 抓取页面，提取最新热门文章
- 筛选：优先选择 AI、全栈、前端相关的文章
- 每篇文章提取：标题、链接、简短概述

### 第三步：整理与输出

将抓取到的内容按以下格式整理输出：

```markdown

- [动态标题](链接)
  - 动态的简单概述

### 天猫

- [动态标题](链接)
  - 动态的简单概述

### 京东

- [动态标题](链接)
  - 动态的简单概述

### 抖音电商

- [动态标题](链接)
  - 动态的简单概述

## 技术

### AI

- [动态标题](链接)
  - 动态的简单概述

### 前端

- [动态标题](链接)
  - 动态的简单概述

### 全栈

- [动态标题](链接)
  - 动态的简单概述
```

## 输出规范

- 电商动态按平台分类（淘宝、天猫、京东、抖音电商），每个平台下优先列出与治理体验相关的规则动态
- 技术动态按领域分类（AI、前端、全栈等），根据实际抓取到的内容归类
- 每个章节总计不超过 13 条动态，优先选择最重要和最新的内容
- 每条动态必须包含可点击的链接和简洁的概述（1-2 句话）
- 如果某个数据源无法访问，在对应分类下标注"暂无数据"并继续处理其他数据源，不要因为单个数据源失败而中断整个流程

## 抓取失败处理

由于部分数据源为 SPA 应用或存在反爬机制，WebFetch 可能无法直接获取内容。遇到这种情况时：

1. 首先尝试 WebFetch 直接抓取
2. 如果返回空内容或错误，使用 WebSearch 搜索相关关键词获取替代信息
3. 搜索关键词示例："淘宝规则更新 2026"、"京东商家规则变动"、"抖音电商新规"等
4. 如果搜索也无法获取有效信息，在输出中标注"暂无数据"
