# 变量-常量-作用域

## 变量

### 声明形式

#### 语法格式

```go
package main

func main() {
	var a int            // 声明 int 变量; 使用 int 零值 0
	var b int = 2        // 声明变量; 显式指定类型和值
	var c, d = 3, "go"   // 声明多个变量; 根据初始值推导类型
	e := true            // 短变量声明; 只能用于函数内部
	println(a, b, c, d, e)
}
```

### 声明块

#### 语法格式

```go
var (
	name  string    // string 零值为空字符串
	count int = 1   // 显式指定类型和值
	ready     = false // 省略类型; 由 false 推导为 bool
)
```

### 短变量声明

#### 概念

- `:=`: 声明并初始化局部变量;
- 重声明规则: 同一作用域内至少有一个新变量;
- 常见用途: 接收函数返回值;

#### 语法格式

```go
func main() {
	x := 1       // 声明 x
	x, y := 2, 3 // y 是新变量; x 被重新赋值
	println(x, y)
}
```

### 赋值

#### 语法格式

```go
func main() {
	x := 1
	x = 2       // 给已声明变量赋新值
	a, b := 10, 20
	a, b = b, a // 多重赋值; 右侧先求值再统一赋给左侧
	println(x, a, b)
}
```

## 零值

### 基本规则

#### 零值表

| 类型 | 零值 |
| ---- | ---- |
| bool | false |
| int / uint / float / complex | 0 |
| string | "" |
| pointer | nil |
| slice | nil |
| map | nil |
| channel | nil |
| function | nil |
| interface | nil |
| struct | 各字段零值组合 |
| array | 各元素零值组合 |

### nil

#### 概念

- nil: pointer、slice、map、channel、function、interface 的零值;
- 非类型值: nil 本身没有默认具体类型;
- 比较限制: 只能与可为 nil 的类型值比较;

#### 语法格式

```go
var p *int = nil     // pointer 零值
var s []int = nil    // slice 零值
var m map[string]int // map 零值为 nil
println(p == nil, s == nil, m == nil)
```

### 零值可用

#### 概念

- 零值可用: 部分类型零值即可直接使用;
- nil slice: 可 `append`, 但不能按索引写入不存在元素;
- nil map: 可读取, 但写入前必须 `make`;

#### 语法格式

```go
package main

import "sync"

func main() {
	var mu sync.Mutex // sync.Mutex 零值表示未加锁状态
	mu.Lock()
	mu.Unlock()
}
```

## 常量

### 声明形式

#### 概念

- const: 编译期确定的不可修改值;
- 常量类型: 布尔、数值、字符串等基础常量值;
- 包级常量: 常用于固定配置值和枚举值;

#### 语法格式

```go
const Pi = 3.14 // 常量值在编译期确定

const (
	StatusOK = 200
	Method   = "GET"
)
```

### 无类型常量

#### 语法格式

```go
const n = 10 // n 为 untyped constant

var i int = n     // n 根据上下文转换为 int
var f float64 = n // n 根据上下文转换为 float64
```

### iota

#### 概念

- `iota`: const block 内的递增计数器;
- 初始值: 每个 const block 从 0 开始;
- 递增时机: 每一行常量声明递增一次;

#### 语法格式

```go
const (
	Sunday = iota // 0
	Monday        // 1; 省略表达式时重复上一行非空表达式
	Tuesday       // 2
)
```

### 位标志

#### 概念

- bit flag: 使用二进制位组合多个布尔状态;
- 位移表达式: `1 << iota` 常用于生成独立标志位;

#### 语法格式

```go
const (
	Read = 1 << iota // 1; 二进制 001
	Write            // 2; 二进制 010
	Exec             // 4; 二进制 100
)
```

## 运算符

### 算术运算符

#### 运算符表

| 运算符 | 描述 |
| ------ | ---- |
| `+` | 加法、字符串拼接 |
| `-` | 减法 |
| `*` | 乘法 |
| `/` | 除法 |
| `%` | 取余, 仅整数 |

#### 语法格式

```go
func main() {
	println(7 / 2)         // 整数除法结果为 3
	println(7 % 2)         // 整数取余结果为 1
	println("go" + "lang") // 字符串拼接
}
```

### 比较运算符

#### 概念

- 比较结果: bool;
- 可比较类型: bool、number、string、pointer、channel、interface、可比较元素组成的 array 和 struct;
- 不可比较类型: slice、map、function;

#### 运算符表

| 运算符 | 描述 |
| ------ | ---- |
| `==` | 相等 |
| `!=` | 不相等 |
| `<` | 小于 |
| `<=` | 小于等于 |
| `>` | 大于 |
| `>=` | 大于等于 |

### 逻辑运算符

#### 语法格式

```go
func main() {
	a := true
	b := false
	println(a && b) // 逻辑与
	println(a || b) // 逻辑或
	println(!a)     // 逻辑非
}
```

### 位运算符

#### 运算符表

| 运算符 | 描述 |
| ------ | ---- |
| `&` | 按位与 |
| `|` | 按位或 |
| `^` | 按位异或或一元按位取反 |
| `&^` | 按位清除 |
| `<<` | 左移 |
| `>>` | 右移 |

#### 语法格式

```go
func main() {
	var flags uint = 0b1010
	println(flags & 0b0010)  // 按位与
	println(flags &^ 0b1000) // 清除指定二进制位
}
```

### 赋值运算符

#### 语法格式

```go
func main() {
	n := 1
	n += 2  // 等价于 n = n + 2
	n <<= 1 // 等价于 n = n << 1
	println(n)
}
```

## 作用域

### 作用域层级

#### 分类

| 作用域 | 描述 |
| ------ | ---- |
| universe | 预定义标识符作用域 |
| package | 同一 package 内可见 |
| file | 当前文件可见 |
| block | 当前代码块可见 |

### universe 作用域

#### 概念

- universe scope: 预定义标识符所在作用域;
- 预定义类型: `int`、`string`、`bool`、`error`;
- 预定义值: `true`、`false`、`nil`;
- 预定义函数: `make`、`new`、`append`、`len`、`cap`、`panic`、`recover`;

### package 作用域

#### 语法格式

```go
package config

var Port = 8080       // 首字母大写; 包外可见
var host = "127.0.0.1" // 首字母小写; 仅包内可见
```

### block 作用域

#### 语法格式

```go
func main() {
	x := 1
	if x > 0 {
		y := 2 // y 只在 if 块内部可见
		println(x, y)
	}
}
```

### 变量遮蔽

#### 概念

- shadowing: 内层作用域声明同名变量, 遮蔽外层变量;
- 风险: 易误以为修改了外层变量;
- 处理方式: 对关键变量避免重复命名;
#### 语法格式

```go
package main

var name = "package"

func main() {
	name := "local" // 声明局部 name; 遮蔽包级 name
	println(name)
}
```
