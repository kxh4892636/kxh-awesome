# 投资中最简单的事

## 核心框架

### 投资本质

#### 实业视角

- 实业眼光: 把股票当作生意所有权评估;
- 客户资金: 按自有资金标准控制风险;
- 长期收益: 来自企业价值增长与错误定价修复;
- 短期波动: 多由情绪、政策、流动性驱动;

#### 三个问题

- 估值: 为什么便宜;
- 品质: 为什么是好公司;
- 时机: 为什么现在买;
- 优先级: 品质最难, 估值最可学, 时机最不可控;

#### 三个工具

- Porter Five Forces: 判断行业门槛、上下游议价权、竞争格局;
- DuPont Analysis: 判断利润率、周转率、杠杆来源;
- Valuation Analysis: 横向同业比较、纵向历史比较、市值成长空间比较;

## 好生意

### 品质判断

#### 商业模式

- 好生意: 容易赚钱、现金流好、可重复扩张;
- 烂生意: 竞争激烈、产品短命、现金流差、定价权弱;
- 门槛: 别人做不了;
- 可复制: 自己能重复做;
- 取舍: 宁要有门槛的低增长, 不要无门槛的高增长;

#### 现金流

- 现金流: 生意质量的最终验证;
- 预收模式: 客户资金支持扩张, 现金流压力低;
- 先投后收: 项目制或内容制业务, 失败一次即可伤筋动骨;
- 利润质量: 现金流长期背离利润时降低可信度;

#### 定价权

- 定价权: 核心竞争力的结果;
- 品牌: 高端消费品常见定价权来源;
- 渠道: 物流、网点、服务网络构成进入壁垒;
- 成本: 同质化行业的低成本优势;
- 规模: 越大越强的行业更易形成龙头优势;

### 行业格局

#### 数月亮

- 月亮: 胜负已分的行业龙头;
- 星星: 高度分散的竞争者;
- 月朗星稀: 一家独大或一超多强的理想格局;
- 胜而后求战: 等竞争格局清晰后买赢家;
- 战而后求胜: 在百舸争流中押注未来赢家;

#### 行业层级

- 细分市场: 大行业分散不等于子行业分散;
- 区域市场: 啤酒、水泥、地产等销售半径行业可形成局部垄断;
- 跨界竞争: 新品类可重新定义原有行业边界;
- 天空定义: 判断公司是否真为局部月亮的前提;

#### 寡头来源

| 来源     | 特征                | 投资含义            |
| -------- | ------------------- | ------------------- |
| 市场竞争 | 洗牌后自然集中;     | 更可能拥有定价权;   |
| 行政许可 | 牌照、管制、配额;   | 垄断未必带来高回报; |
| 网络效应 | 用户越多体验越好;   | 容易赢家通吃;       |
| 规模经济 | 单位成本随规模下降; | 龙头利润率持续领先; |

## 便宜

### 估值原则

#### 便宜是硬道理

- 低估值: 安全边际的主要来源;
- 好价格: 弥补判断误差和意外冲击;
- 高估值: 透支未来, 容易估值与盈利双杀;
- 普通公司: 不应支付伟大公司的价格;
- 低预期: 更容易出现业绩超预期;

#### 方法有效性

- 长期有效: 不要求每年有效;
- 短期落后: 方法未失效的必要代价;
- 风格漂移: 用短期结果否定长期方法的常见错误;
- 过程正确: 需跨越 5-10 年检验;

### 买入时机

#### 逆向投资

- 逆向本质: 品格多于技巧;
- 人弃我取: 在好公司被短期问题抛弃时介入;
- 估值前提: 坏消息已被充分反映;
- 问题性质: 短期、可解决、非永久性;
- 反身性: 股价下跌不应反过来毁坏公司基本面;

#### 分批建仓

- 左侧买入: 容易买早;
- 负面钝化: 坏消息继续出现但股价不再下跌;
- 越跌越买: 仅适用于价值易估、无反身性、低估值高品质资产;
- 收缩战线: 市场下跌时集中到研究最透、估值最低、竞争优势最强的标的;

## 风险

### 真假风险

#### 风险类型

| 类型     | 本质                  | 应对                        |
| -------- | --------------------- | --------------------------- |
| 感受风险 | 投资者主观恐惧;       | 与真实风险分开判断;         |
| 真实风险 | 本金永久性损失概率;   | 重点规避;                   |
| 暴露风险 | 市场已充分恐惧并折价; | 可承担并获取补偿;           |
| 隐藏风险 | 市场尚未定价的危险;   | 避免承担;                   |
| 波动风险 | 价格短期剧烈变化;     | 用期限、仓位、产品结构吸收; |
| 永久损失 | 内在价值不可逆受损;   | 用安全边际和止错规避;       |

#### 安全边际

- 多情景收益: 东方不亮西方亮;
- 低估值保护: 估错或受损后仍不亏本金;
- 冗余设计: 保留备用系统, 避免零乘任何数;
- 价值易估: 业务简单、现金流稳定、估值锚清晰;
- 无反身性: 股价下跌不影响经营、生存和融资;

### 卖出规则

#### 止错

- 基本面恶化: 公司没有想象中好;
- 估值修复: 价格不再便宜;
- 机会成本: 出现更好更便宜的资产;
- 买入成本: 不构成卖出理由;
- 止损区别: 趋势投资止损, 价值投资止错;

## 陷阱

### 价值陷阱

#### 类型

- 技术淘汰: 胶卷、功能机、旧技术路径被替代;
- 赢家通吃小公司: 龙头之外价值被压缩;
- 分散重资产夕阳行业: 需求不增、产能难退、价格战长期化;
- 周期顶点低 PE: 顶峰利润不可持续;
- 会计欺诈: 报表利润与真实现金不一致;
- 人不行: 好资产被差组织或差管理消耗;

#### 判断

- 黄金坑: 短期因素导致便宜, 内在逻辑未破坏;
- 融化冰棒: 永久性原因导致便宜, 价值随时间消失;
- 低估修复: 估值从高估回归合理, 不等于便宜;
- 逻辑破坏: 护城河、竞争力、行业结构发生不可逆恶化;

### 成长陷阱

#### 类型

- 估值过高: 高预期带来高失望概率;
- 技术路径踏空: 标准或路线失败导致投入归零;
- 无利润增长: 烧钱不能形成黏性、网络效应、规模优势;
- 成长性破产: 扩张越快现金缺口越大;
- 盲目多元化: 为维持增速偏离主业;
- 树大招风: 无门槛成功吸引竞争者;
- 新产品风险: 投入确定、回报不确定;
- 寄生式增长: 命脉掌握在大客户或平台手中;
- 强弩之末: 高增长已过峰值但估值仍高;
- 会计造假: 高增长压力放大造假动机;

## 周期

### 行业轮动

#### 四种周期

- 政策周期: 通常领先市场底;
- 市场周期: 通常领先经济底;
- 经济周期: 通常领先盈利底;
- 盈利周期: 通常最后确认复苏;

#### 三种杠杆

- 财务杠杆: 利率下降时最敏感;
- 运营杠杆: 收入小幅恢复带来利润大幅反弹;
- 估值杠杆: 流动性充裕时想象空间放大估值;

#### 阶段策略

- 熊末牛初: 看政策与资金面, 不等微观盈利修复;
- 经济复苏: 高运营杠杆行业弹性最大;
- 牛市后段: 估值杠杆主导, 价值投资容易提前离场;
- 强周期行业: 可做波段, 不宜当作长期价值持有;
- 立足点: 时机辅助判断, 估值与品质才是可把握变量;

## 心理误区

### 行为偏差

#### 常见误区

- 野花偏好: 高估未来成长, 忽视现有价值;
- 过度自信: 默认自己显著强于平均水平;
- 仓位思维: 持仓后只收集支持性证据;
- 锚固偏见: 用历史价格替代估值判断;
- 过度外推: 把短期趋势当作长期趋势;
- 亏损厌恶: 盈利早卖、亏损死扛;
- 标题党: 对新闻标题过度反应;
- 榔头症: 用单一方法套所有行业;
- 选择性记忆: 记住成功, 忘记错误;
- 差点就赢: 被接近成功诱导重复下注;
- 羊群效应: 集体一致时风险最大;
- 心理账户: 把买入成本和浮盈浮亏纳入决策;

### 市场叙事

#### 后视镜

- 后视镜效应: 用上一轮经验指导下一轮市场;
- 风格追逐: 今年做去年应该做的事;
- 资产轮动: 10 年级别趋势不可简单外推;
- 前瞻判断: 跑向球将要去的位置;

#### 这次不同

- 周期本质: 经济与市场反复起落;
- 高点叙事: 用新时代解释高估值;
- 低点叙事: 用长期悲观解释低估值;
- 历史训练: 多读旧书、旧报告、旧危机;

#### 树风心

- 心动: 短期情绪博弈;
- 风动: 中期政策与流动性;
- 树动: 长期基本面;
- 零和游戏: 研究情绪赚对手的钱;
- 正和游戏: 研究企业赚成长的钱;

## 操作清单

### 买入前

#### 检查项

- 行业格局: 是否月朗星稀或正在形成月亮;
- 商业模式: 是否容易赚钱且现金流好;
- 定价权: 是否来自品牌、渠道、成本、规模、牌照或网络效应;
- 估值: 是否足够反映大多数坏情况;
- 问题性质: 是否短期、可解决、非永久性;
- 反身性: 股价下跌是否会伤害基本面;
- 安全边际: 是否存在多情景收益和冗余保护;
- 能力圈: 是否能解释公司如何赚钱、为何赚钱、为何能持续赚钱;

### 持有中

#### 检查项

- 基本面: 原投资逻辑是否被破坏;
- 竞争格局: 龙头优势是否扩大或缩小;
- 现金流: 利润是否能转化为现金;
- 估值: 价格是否仍低于保守内在价值;
- 替代机会: 是否出现更便宜、更好、更确定资产;
- 心理偏差: 是否因仓位、成本、浮盈浮亏影响判断;

### 卖出时

#### 检查项

- 止错: 公司没有想象中好;
- 止盈: 估值已充分反映价值;
- 换仓: 有更好更便宜选择;
- 避险: 出现反身性、永久损失、造假、技术淘汰信号;
- 禁忌: 不因买入成本、近期涨跌、新闻标题单独决策;
