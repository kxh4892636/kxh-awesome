# freeform-drawing 验收 Prompt

这些 prompt 用于验收自由 SVG 路径，不是模板。验证时必须重新生成 `diagram.svg`，不得复制旧 SVG 坐标或节点。

## 1. 流量归因桑基图

画一个“增长流量归因桑基图”：左侧是自然搜索、内容推荐、付费投放、社群转介绍；中间是首次访问、注册、激活、付费；右侧是留存用户和流失用户。用流带粗细表达相对贡献，自然搜索和内容推荐贡献较高，付费投放转化较低但流量稳定。输出到飞书画板。

验收重点：

- mode=`sankey`
- 至少 4 条宽度不同的流带
- 来源、中间阶段、结果三段关系一眼能看懂
- 不是普通流程图，也不是等宽箭头图

## 2. 产品路线图

画一个“AI 运维助手产品路线图”：按 2026 Q2、Q3、Q4 三阶段展示，泳道包含数据接入、诊断能力、协同闭环。里程碑包括日志接入、告警归因、变更关联、Runbook 推荐、群内协作、自动复盘。输出到飞书画板。

验收重点：

- mode=`roadmap`
- 有时间轴、阶段泳道和里程碑
- 有少量真实依赖线
- 不是横向卡片堆叠

## 3. 雨后山林清晨插画

画一幅“雨后山林清晨”纯 SVG 矢量插画：远处山脉、层叠树林、低雾、清晨太阳、几只飞鸟、湿润空气感。输出到飞书画板。

验收重点：

- mode=`illustration`
- 有前景/中景/背景层次
- 有雾气、太阳、飞鸟等氛围元素
- 色彩来自 house_style 低饱和色，用透明叠层形成层次
