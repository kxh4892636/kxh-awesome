# 控制流

## if

### 基本条件

#### 语法格式

```go
func main() {
	score := 80
	if score >= 60 { // if 条件必须是 bool; 条件不需要括号
		println("pass")
	} else { // else 必须跟在前一个右花括号同一行
		println("fail")
	}
}
```

### 初始化语句

#### 语法格式

```go
func main() {
	if n := len("go"); n > 0 { // n 只在 if-else 内可见
		println(n)
	}
}
```

### 多分支

#### 语法格式

```go
func main() {
	n := 0
	if n > 0 {
		println("positive")
	} else if n < 0 { // 从上到下匹配第一个 true 分支
		println("negative")
	} else {
		println("zero")
	}
}
```

## switch

### 表达式 switch

#### 概念

- 自动 break: 每个 case 默认不会继续执行下一个 case;
- default: 可放在任意 case 位置, 通常放在最后;

#### 语法格式

```go
func main() {
	day := "Mon"
	switch day {
	case "Sat", "Sun": // 多值 case 使用逗号分隔
		println("weekend")
	case "Mon":
		println("start")
	default: // 没有 case 匹配时执行
		println("workday")
	}
}
```

### 无表达式 switch

#### 语法格式

```go
func main() {
	n := 10
	switch { // 等价于 switch true
	case n < 0:
		println("negative")
	case n == 0:
		println("zero")
	default:
		println("positive")
	}
}
```

### fallthrough

#### 概念

- `fallthrough`: 不能用于最后一个 case;
- 使用建议: 少用, 避免破坏 switch 默认清晰语义;

#### 语法格式

```go
func main() {
	n := 1
	switch n {
	case 1:
		println("one")
		fallthrough // 强制继续执行下一个 case
	case 2:
		println("next")
	}
}
```

### type switch

#### 语法格式

```go
package main

import "fmt"

func printType(v any) {
	switch x := v.(type) { // v.(type) 只能用于 type switch
	case int:
		fmt.Println("int", x) // x 在此分支中为 int
	case string:
		fmt.Println("string", x) // x 在此分支中为 string
	default:
		fmt.Println("unknown")
	}
}
```

## for

### 三段式循环

#### 语法格式

```go
func main() {
	for i := 0; i < 3; i++ { // init; condition; post
		println(i)
	}
}
```

### 条件循环

#### 语法格式

```go
func main() {
	n := 3
	for n > 0 { // condition-only for; 表达类似 while 的循环
		println(n)
		n--
	}
}
```

### 无限循环

#### 语法格式

```go
func main() {
	for { // 无限循环
		println("once")
		break // 退出当前循环
	}
}
```

## range

### 遍历 slice

#### 语法格式

```go
func main() {
	nums := []int{10, 20, 30}
	for i, v := range nums { // i 为下标; v 为元素值副本
		println(i, v)
	}
}
```

### 遍历 map

#### 语法格式

```go
func main() {
	m := map[string]int{"a": 1, "b": 2}
	for k, v := range m {
		println(k, v) // map 遍历顺序不保证稳定
	}
}
```

### 遍历 string

#### 语法格式

```go
func main() {
	for i, r := range "语言" {
		println(i, r) // i 为 rune 起始字节位置; r 为当前 Unicode code point
	}
}
```

### 忽略返回值

#### 语法格式

```go
func main() {
	nums := []int{1, 2, 3}
	for _, v := range nums { // _ 丢弃不需要的 index
		println(v)
	}
}
```

### 循环变量

#### 概念

- range value: 每轮迭代得到的是元素值副本;
- 修改元素: 需要通过 index 写回原集合;
- 地址使用: 需要区分循环变量地址和元素地址;

#### 语法格式

```go
func main() {
	nums := []int{1, 2, 3}
	for i, v := range nums {
		nums[i] = v * 10 // 通过 index 修改原 slice 元素
	}
}
```

## break 和 continue

### 基本用法

#### 语法格式

```go
func main() {
	for i := 0; i < 5; i++ {
		if i == 2 {
			continue // 跳过当前轮剩余代码
		}
		if i == 4 {
			break // 终止当前 for
		}
		println(i)
	}
}
```

### 标签

#### 概念

- label: 标记一条语句位置;
- 使用场景: 退出多层嵌套循环;
#### 语法格式

```go
func main() {
outer:
	for i := 0; i < 3; i++ {
		for j := 0; j < 3; j++ {
			if i*j > 2 {
				break outer // 跳出 outer 标签对应的外层循环
			}
		}
	}
}
```
