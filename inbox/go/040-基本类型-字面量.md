# 基本类型-字面量

## 类型系统

### 基本概念

#### 定义

- type: 值的模板, 决定存储形式和可用操作;
- value: 某个类型的具体数据;
- literal: 直接写在源码中的值表示;
- conversion: 显式类型转换, 语法为 `T(v)`;

### 内置类型

#### 类型表

| 分类 | 类型 |
| ---- | ---- |
| 布尔 | bool |
| 字符串 | string |
| 有符号整数 | int, int8, int16, int32, int64 |
| 无符号整数 | uint, uint8, uint16, uint32, uint64, uintptr |
| 浮点数 | float32, float64 |
| 复数 | complex64, complex128 |
| 字节 | byte |
| 字符码点 | rune |

## bool

### 布尔值

#### 语法格式

```go
func main() {
	var ok bool // bool 零值为 false
	ok = true   // bool 只能保存 true 或 false
	println(ok)
}
```

## 整数

### 整数类型

#### 类型表

| 类型 | 描述 |
| ---- | ---- |
| int | 平台相关有符号整数 |
| uint | 平台相关无符号整数 |
| int8 / int16 / int32 / int64 | 固定位宽有符号整数 |
| uint8 / uint16 / uint32 / uint64 | 固定位宽无符号整数 |
| byte | uint8 别名 |
| rune | int32 别名 |
| uintptr | 可保存指针位模式的整数 |

### 整数字面量

#### 语法格式

```go
func main() {
	decimal := 42        // 十进制
	binary := 0b101010   // 二进制
	octal := 0o52        // 八进制
	hex := 0x2a          // 十六进制
	readable := 1_000_000 // 下划线仅提升可读性
	println(decimal, binary, octal, hex, readable)
}
```

### 整数溢出

#### 语法格式

```go
package main

import "fmt"

func main() {
	var n uint8 = 255 // uint8 范围为 0..255
	n++              // 溢出后回绕为 0
	fmt.Println(n)
}
```

## 浮点数

### 浮点类型

#### 类型表

| 类型 | 描述 |
| ---- | ---- |
| float32 | 32 位浮点数 |
| float64 | 64 位浮点数 |

### 浮点字面量

#### 语法格式

```go
func main() {
	a := 1.5     // 小数字面量; 默认推导为 float64
	b := 3.125e2 // 科学计数法; 等于 312.5
	c := 0x1p-2  // 十六进制浮点; 等于 0.25
	println(a, b, c)
}
```

### 精度

#### 概念

- 精度误差: 十进制小数不一定能被浮点数精确表示;
- 金额计算: 不应直接依赖 float 精确表示;

#### 语法格式

```go
package main

import "fmt"

func main() {
	fmt.Println(0.1 + 0.2) // 浮点数使用二进制近似表示十进制小数
}
```

## 复数

### 复数类型

#### 语法格式

```go
func main() {
	var z complex128 = 1 + 2i // complex128 由两个 float64 组成
	println(real(z), imag(z)) // real 取实部; imag 取虚部
}
```

## string

### 字符串类型

#### 语法格式

```go
func main() {
	s := "hello"  // string 为不可变字节序列
	println(len(s)) // len 返回底层字节数
}
```

### 内部存储

#### 概念

- string 底层: 只读字节序列;
- 编码约定: Go 源码中的字符串字面量通常为 UTF-8;
- 索引结果: `s[i]` 返回 byte, 不返回 rune;
- 修改限制: string 不可原地修改;

#### 语法格式

```go
func main() {
	s := "语言"
	println(len(s)) // 输出字节数, 不是字符数量
	println(s[0])   // 返回第 0 个 byte
}
```

### 字符串比较

#### 概念

- string 可比较: 可使用 `==`、`!=`、`<`、`<=`、`>`、`>=`;
- 比较方式: 按字节序列逐字节比较;
- 排序语义: 不等于自然语言排序;

#### 语法格式

```go
func main() {
	println("go" == "go") // true
	println("abc" < "abd") // true; 按字节序比较
}
```

### 转义字符串

#### 语法格式

```go
func main() {
	s := "line1\nline2" // 双引号字符串处理反斜杠转义
	println(s)
}
```

### 原始字符串

#### 语法格式

```go
func main() {
	path := `C:\tmp\file.txt` // 反引号字符串不处理反斜杠转义
	println(path)
}
```

## rune

### 字符码点

#### 语法格式

```go
package main

import "fmt"

func main() {
	var r rune = '中'     // rune 是 int32 别名, 表示 Unicode code point
	fmt.Printf("%c %U\n", r, r) // %c 显示字符; %U 显示 Unicode 码点
}
```

### 字符字面量

#### 语法格式

```go
func main() {
	a := 'a'        // rune literal
	newline := '\n' // 转义字符
	han := '\u4e2d' // Unicode code point
	println(a, newline, han)
}
```

### 字符串遍历

#### 概念

- `range string`: 按 UTF-8 解码后遍历 rune;
- 字符串索引: 按 byte 访问, 不按字符访问;

#### 语法格式

```go
package main

import "fmt"

func main() {
	for index, r := range "Go语言" {
		fmt.Println(index, r, string(r)) // index 为起始字节位置; r 为当前 rune
	}
}
```

## 类型转换

### 数值转换

#### 概念

- 隐式转换: Go 不做数值类型间隐式转换;
- 窄类型转换: 可能截断数据;

#### 语法格式

```go
func main() {
	var a int = 10
	var b int64 = int64(a)     // 显式转换 int 为 int64
	var c float64 = float64(a) // 显式转换 int 为 float64
	println(b, c)
}
```

### string 与 byte slice

#### 概念

- string 不可变: 修改 byte slice 不会修改原字符串;
- 转换复制: `string` 与 `[]byte` 转换通常产生新存储;
#### 语法格式

```go
func main() {
	s := "go"
	bs := []byte(s) // 复制字符串字节到 byte slice
	bs[0] = 'G'
	s2 := string(bs) // 复制 byte slice 到新字符串
	println(s, s2)
}
```
